# GitHub Actions Firebase Deploy Implementation Plan> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (recommended) or superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.**Goal:** Automatically deploy the SkillForge Portal to Firebase Hosting and Firestore Rules whenever `main` is updated.**Architecture:** A GitHub Actions workflow runs tests (`npm ci` + `npm test`) and, on success, deploys `hosting` + `firestore:rules` to the Firebase project defined in `.firebaserc`.**Tech Stack:** GitHub Actions, Node.js, Firebase CLI (`firebase-tools`), Firebase Service Account JSON (GitHub Secret)---## Pre-Reqs (Owner Setup)- Create a Firebase Service Account key for the Firebase project:  - Project ID: `skillfoge-ecosystem`  - Required permissions: Firebase Hosting Admin + Firestore Admin (or Editor for the project).- Add the JSON key to GitHub repo secrets:  - Secret name: `FIREBASE_SERVICE_ACCOUNT_SKILLFOGE_ECOSYSTEM`  - Value: the full JSON content (single-line or multi-line is fine).## File Map**Create**- `.github/workflows/firebase-deploy.yml`---### Task 1: Add workflow for deploy on main**Files:**- Create: `.github/workflows/firebase-deploy.yml`- [ ] **Step 1: Create the workflow file**Create `.github/workflows/firebase-deploy.yml` with:```yamlname: Deploy to Firebase (Portal)on:  push:    branches: [ main ]concurrency:  group: firebase-deploy-portal-main  cancel-in-progress: truejobs:  test-and-deploy:    runs-on: ubuntu-latest    steps:      - name: Checkout        uses: actions/checkout@v4      - name: Setup Node        uses: actions/setup-node@v4        with:          node-version: 20          cache: npm          cache-dependency-path: package-lock.json      - name: Install deps        run: npm ci      - name: Run tests        run: npm test      - name: Deploy Hosting        uses: FirebaseExtended/action-hosting-deploy@v0        with:          repoToken: ${{ secrets.GITHUB_TOKEN }}          firebaseServiceAccount: ${{ secrets.FIREBASE_SERVICE_ACCOUNT_SKILLFOGE_ECOSYSTEM }}          projectId: skillfoge-ecosystem          channelId: live          entryPoint: .      - name: Write service account JSON        run: |          echo '${{ secrets.FIREBASE_SERVICE_ACCOUNT_SKILLFOGE_ECOSYSTEM }}' > /tmp/firebase-sa.json          chmod 600 /tmp/firebase-sa.json        shell: bash      -
